## 🎯 核心洞察彙整

### 洞察 1：功能優先級（轉換率影響力排序）

| 排名 | 功能 | 影響力 | 原因 |
|------|------|--------|------|
| 🥇 | **智慧課程預覽** | 極高 | 直接消除「資訊不對稱」，降低購買風險感 |
| 🥈 | **購買決策助手** | 高 | 臨門一腳，解決結帳頁猶豫 |
| 🥉 | **AI 購物顧問** | 高 | 流量入口，自然語言導流高意圖用戶 |
| 4 | 動態雷達圖 | 中 | 增加趣味性，但手動調權重用戶不會用 |
| 5 | 學習路徑規劃 | 中 | 提高 LTV，但初期效果不明顯 |
| 6 | 需求探索器 | 低 | 用戶沒耐心跟機器人聊十句 |
| 7 | 全球比較器 | 低 | 高成本高風險（爬蟲法律問題） |
| 8 | 學習成效追蹤 | 低 | 購買後功能，不影響轉換 |

### 洞察 2：開發陷阱警告

```
⚠️ 全球課程比較器
   - 爬取 Udemy/Coursera 有反爬蟲和法律風險
   - 替代方案：用戶貼連結 → AI 分析該頁面
   - 或開發 Chrome Extension 讓用戶主動抓取

⚠️ 互動式需求探索器
   - 過度對話增加摩擦力
   - 應合併入 AI 購物顧問，只在搜尋結果太發散時才跳出

⚠️ 動態雷達圖權重調整
   - 「看起來很炫但沒人用」
   - 讓 AI 自動生成，保留手動調整給 Power User
```

### 洞察 3：BYOK 策略

**結論：BYOK 放在 Free/低價層，而非 Pro 功能**

理由：
1. 成本轉嫁 — 讓免費用戶自己付 LLM Token 費用
2. 你的價值在「Workflow」— 不是轉售 AI，而是 MVT 驗證邏輯、Pareto 演算法
3. 吸引力 — 重度用戶更喜歡 BYOK，不受平台 Rate Limit 限制

## 💰 訂閱定價策略（確定版）

### 三層定價設計

| Tier | 價格 | 功能 | 目標用戶 |
|------|------|------|----------|
| **Starter** | **$8/月** | BYOK 無限分析、購買決策助手、進階預覽、動態雷達圖 | BYOK 技術用戶 |
| **Pro** | **$20/月** | +500 Managed Credits、URL 課程分析、PDF 匯出、學習路徑規劃 | 不想管 API 的一般用戶 |
| **Team** | **$49/月** | +2000 Credits、3 團隊座位、API 存取、白標選項、優先客服 | 小型團隊/Agency |

> 📌 價格階梯：$8 → $20 → $49（約 2.5x 遞增，符合 SaaS 定價心理學）

### 防止「買完就跑」

1. **AI 額度限制** — LTD 包含每月固定 Credit，用完需額外購買
2. **社群權限** — LTD 可進入 Discord，但「即時教練」需付費
3. **持續更新** — 新功能優先給活躍用戶測試

### 文案重點

> **Headline**: "Stop Guessing. Sell Digital Products Validated by AI."
>
> **Key Benefit**: 強調 MVT Report — 在你投入廣告費之前，AI 先告訴你產品的「市場存活率」

## ⚠️ 風險評估

| 風險 | 嚴重度 | 緩解策略 |
|------|--------|----------|
| **AI Token 成本失控** | 🔴 高 | 設定每日/月呼叫上限、使用 Gemini Flash 降成本 |
| **兌換碼濫用** | 🟡 中 | 每 Code 只能兌換一次、限制單帳號疊加數 |
| **儲存成本** | 🟡 中 | 限制上傳 500MB、自動轉移舊檔至冷儲存 |

## 📈 上線 KPI

### 北極星指標
**「成功生成的 AI 分析報告數量」**

### 正向信號 ✅
- 訪客 → 註冊 → 執行第一次 AI 分析 > **20%**
- 獲得報告後 72 小時內完成上架 > **30%**

### 警示信號 🚨
- 有流量但零成交 → 檢查定價或支付流程
- 少數用戶消耗大量 Token 無產出 → 調整 Rate Limit

## 🏁 競爭定位

### 藍海策略

> **不做「賣課平台」，做「數位資產孵化器」**

| 平台 | 定位 | Digital Gateway 機會 |
|------|------|---------------------|
| Udemy | 低價紅海 | 避開純影音，專注高單價數位資產 |
| Hahow | 網紅精品 | 賦能素人，用 AI 數據證明市場 |
| PressPlay | 訂閱壓力 | 買斷制，強調「買了就解決問題」|

### 差異化賣點

1. **BYOK** — 用戶自帶 Key，成本透明
2. **AI 驗證** — 銷售前先驗證產品市場性
3. **全球比較** — 貼連結即時分析外部課程
